# Pharmacy Analytics Prototype

A lightweight, data‑driven dashboard that visualizes pharmacy sales and inventory trends using Python, Pandas, and Plotly.

## Features
- Top‑selling medicines (Units Sold)
- Revenue contribution by product
- Inventory levels with low‑stock alerts
- Auto‑generated interactive dashboard (HTML)

## How to Run
Run the script: 
\`\`\`
python app.py
\`\`\`

This generates:

\`\`\`
docs/index.html
\`\`\`

which can be published via GitHub Pages.

## Tech Stack
- Python
- Pandas
- Plotly
- GitHub Pages

## Status
